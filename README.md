# Projeto Api CRUD - MySQL

## Iniciar projeto
 - Primeiro baixe as dependência
 
 ```bash
  npm install
  #or
  yarn
 ```

 - Depois é so iniciar

 ```bash
  npm run start
  #or
  yarn start
 ```
 - Inicie seu banco de dados na porta 3006
 
 - O projeto estará rodando na porta 8000
