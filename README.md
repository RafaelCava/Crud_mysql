# Projeto Api CRUD - MySQL

## Iniciar projeto
 - Primeiro baixe as dependência
 
 ```bash
  npm install
  #or
  yarn
 ```

 - Depois é so iniciar

 ```bash
  npm run start
  #or
  yarn start
 ```
 - O projeto estará rodando na porta 8000
